# API
### api(Application Programming Interface)
api는 컴퓨터나 소프트웨어를 서로 연결한다.
예를 들면 손님(프로그램)이 웨이터(API)에게 주문을하면 주방(API 제공자, 공공포탈등등)에게 주문을 전달하고, 주방에서 다시 웨이터(API)가 주문을 음식을 가져온다.
API를 사용하는 방법과 용도에 따라

### value
- 함수가 호출될 때, 메모리 공간 안에서는 함수를 위한 임시공간이 생성됨, 함수가 종료되면 해당 공간은 사라진다
- Stack Frame : 함수 호출 시 할당되는 메모리 블록
- 복사된 인자는 함수 안에서 지역적으로 사용되는 local value의 특성을 가짐(복사되는 것)
- 즉, 매개변수를 변경해도 원래의 변수에는 영향을 미치지 않음 

###reference
- value랑 같이 임시공간 생성, 함수가 종료되면 사라짐
- 참조에 의한 호출방식은 함수 호출 시 인자로 전달되는 변수의 레퍼런스를 전달한다(변수를 가르킴)
- value와 다르게 복사되는 것이 아니여서, 함수 안에서 인자 값이 변경되면, 아규먼트로 전달되어 객체의 값도 변경됨
